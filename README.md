
##SwiftColorPlugin
专门用于Swift的Xcode颜色插件，可以在编写代码的时候预览和修改颜色代码，目前只识别如下格式的颜色代码
<br>
"#ffffff".toColor
<br>
或者
<br>
"#ffffffff".toColor
<br>
需要自行写一个String转UIColor的扩展，这个扩展的实现可以参考 https://github.com/xiyuan-fengyu/IOSSwiftDemos/blob/master/IOSSwiftDemos/IOSSwiftDemos/StringUtil.swift
